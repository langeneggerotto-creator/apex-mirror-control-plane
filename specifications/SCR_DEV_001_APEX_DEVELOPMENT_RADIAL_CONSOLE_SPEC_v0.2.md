## 2. Purpose and Operator Outcome

### Product Promise

> Provide one human-controlled development console where every active APEX module can be opened, understood, compared, directed, evidenced and advanced only through verified progress.

### Operator Must Be Able To

1. see all six active modules around a common human-control center;
2. select a module pod and reveal its purpose, features, truth status, repository, gaps, evidence and direction stack;
3. change among six useful interface views without losing current context;
4. see whether a module is in `VERIFIED_VERTICAL_SLICE_FIRST` mode and whether scale is locked or eligible;
5. select `STAY`, `SWITCH`, `HOLD`, `STOP` or `INNOVATE` as a bounded direction decision;
6. export one approved task packet and one evidence-receipt candidate for preservation.

## 6. Concentric Progress Rings

The rings are not decoration. Each ring exposes a distinct operational status.

| Ring | Label | Segments | Function |
|---:|---|---|---|
| `R1` | Development Stage Ring | Observe, Define, Model, Build, Verify, Preserve, Decide, Improve | Identifies where each selected module currently sits in its build path. |
| `R2` | Truth Status Ring | Specified, Prototype, Tested, Blocked, Aspirational, Verified-in-Scope | Displays claim maturity without overstatement. |
| `R3` | Evidence Ring | Artifact, Test, Commit, Limitation, Approval, Receipt | Shows evidence coverage and missing proof. |
| `R4` | Direction / Benefit Ring | Stay, Switch, Hold, Stop, Innovate, Scale Review | Supports operator decision and benefit-governor status. |
| `R5` | Scale Eligibility Ring | VS Defined, VS Running, VS Verified, Scale-10, Scale-100, APEX-1M Review | Makes scaling conditional on slice proof. |

### Ring Behavior

- Hovering a segment surfaces its definition and status.
- Selecting a module causes rings to display that module‚Äôs state.
- Selecting the Human Control Core causes rings to summarize portfolio state.
- Missing evidence must display as an explicit gap, not as zero-value or hidden content.
